NightShift NPC System
📖 Description
NightShift is an automated NPC spawning system for RedM servers. NPCs spawn at configured locations during night hours, walk to their designated positions, and leave when day arrives. Perfect for creating lively saloons, bars, and gathering spots that feel alive at night.

✨ Features
✅ Automatic NPC spawning at night
✅ NPCs walk in from entrance to destination
✅ NPCs walk back to entrance and despawn at day
✅ Configurable spawn times
✅ Multiple locations support
✅ Multiple NPC models support
✅ Performance friendly
✅ Debug mode for testing
